# 使用示例

本目录包含 Auto-Coder 的使用示例和演示项目。

## 快速开始

### 示例 1: 创建简单的 Web 应用

1. 在 `todo/simple-web-app/` 目录下创建需求文档 `todo.md`：

```markdown
# 简单 Web 应用需求

创建一个基于 Flask 的简单 Web 应用，包含：
- 主页显示 "Hello World"
- 一个关于页面
- 基本的 CSS 样式
```

2. 配置 `config.json`，设置 `PROJECT_NAME` 为 `simple-web-app`

3. 运行：
```bash
python main.py
```

### 示例 2: 数据分析脚本

1. 创建需求文档描述数据分析任务
2. 系统将自动生成相应的 Python 脚本
